Features

Cross-Platform

Works seamlessly on Windows, Linux, and macOS. Supports both Intel and ARM architectures.

Safe & Reliable

Automatic backups before any changes. Dry-run mode to preview changes before applying.

Fast Performance

Written in Go for maximum performance. Efficiently handles large repositories.

Easy to Use

Simple commands: diagnose, fix, verify. Beautiful CLI with colored output.

Detailed Logging

Comprehensive logs and reports. Complete audit trail of all operations.

Comprehensive Fixes

Fixes null SHA references, missing commits, broken trees, and corrupted objects.

Quick Installation

# Move to a directory in your PATH
# Or run directly: .\nsha.exe --help
# Download the binary
chmod +x nsha-linux-amd64
sudo mv nsha-linux-amd64 /usr/local/bin/nsha
nsha --help
# Download the binary
chmod +x nsha-darwin-amd64
sudo mv nsha-darwin-amd64 /usr/local/bin/nsha
nsha --help
